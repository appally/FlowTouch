给 Mac 的触控板手势规则系统

把触控板，变成 Mac 的指令层。

FlowTouch 用三指、四指和捏合手势完成窗口布局、切屏、桌面空间、 截图和快捷动作。它不只是贴窗工具，而是一套可按应用切换的手势规则系统。

默认就顺手

内置 Starter 和 Full 两套预设。默认三指起步，先避开双指滚动冲突，再逐步扩展。

同一手势，按应用切换

规则支持全局和应用范围。同一个手势，在浏览器、Finder 和设计工具里可以做不同的事。

它不是黑盒

学习模式、吸附预览、状态反馈和系统手势冲突提醒都会明确告诉你它识别到了什么。

从轻量到重度

先解决贴窗，再接管整套桌面流。

先用三指左右上解决最常用的窗口整理。需要时，再继续加入点击、捏合、 切屏、桌面空间、截图、媒体和自定义快捷动作。

安装到你的 Mac 上。

当前版本为 1.0，最低支持 macOS 14.6。 推荐在带有内置触控板的 MacBook 或 Magic Trackpad 设备上使用。 首次授权后，它可以常驻菜单栏，随时接管你的触控板工作流。
